# SpendWise - Expense Tracker App

SpendWise is a powerful and beautifully designed expense tracking app built with Flutter.  
It features dynamic UI/UX, category filters, statistics, and smooth animations to make managing personal finance easy and enjoyable.

- Dark/Light theme toggle
- Category filter chips and color-coded icons
- Add, edit, and delete expenses
- Fast real-time search functionality
- Interactive statistics with visual charts
- Premium Material Design 3 UI
- Undo deletion and advanced gestures
- Optional notes and expense details
- Ready for Android, iOS, Desktop, and Web
